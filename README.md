# LeetCode Solutions

```bash
lectCode/
├── src/
│   ├── main/
│   │   ├── java/ru/edu/lectcode/
│   │   │   ├── easy/

│   │   │   │   └── ...
│   │   │   ├── medium/
│   │   │   ├── hard/

│       ├── java/ru/edu/lectcode/
│       │   ├── easy/

│       │   │   └── ...
│       │   ├── medium/
│       │   │   └── ...
│       │   ├── hard/
│       │       └── ...
├── target/
├── .gitignore

```
